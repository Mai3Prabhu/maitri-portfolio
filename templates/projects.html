Insightify: Data Analysis Platform

This is a Flask application for comprehensive EDA, Feature Engineering, and ML model training with an AI Assistant. It was developed to address the inefficiency of using separate scripts and tools for data science tasks like EDA and preprocessing by making the data science workflow faster.

Core Tech

Python (Flask, Pandas, scikit-learn), HTML, Tailwind CSS, JavaScript, Google Gemini API.

Key Feature

Automated EDA, ML training, and Gemini AI assistance.

Problem Solved

Eliminated the need for disparate tools and extensive scripting for initial data analysis.
